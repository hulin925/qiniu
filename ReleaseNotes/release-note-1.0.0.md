# QNRTCWeb v1.0.0 发布

## 简介
QNRTCWeb 是七牛云推出的一款适用于 Web 平台的实时音视频 SDK，提供了包括音视频通话、静音、发布、订阅多种功能，提供灵活的接口，支持高度定制以及二次开发。

## 功能

* 增加屏幕共享功能
* 增加外部音频导入功能
* 支持 360 浏览器

## 缺陷

* 修复发布过程导致的 ICE 失败
* 修复重复发布的错误
* 修复加入房间中 WS 断连导致无法重连


## 优化
* 增加连麦过程中 p2p 连接可用性, 增加重连策略
